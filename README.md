# 🎉 Telegram Giveaway Bot

Бот для розыгрышей в Telegram-канале. Участники должны быть подписаны на канал.  
Хостинг — **бесплатно** через GitHub Actions.

---

## 🚀 Быстрый старт

### 1. Создать бота в Telegram

1. Откройте [@BotFather](https://t.me/BotFather) → `/newbot`
2. Введите имя и username бота
3. Скопируйте **токен** (вида `123456:ABC-DEF...`)

### 2. Получить свой Telegram ID

Напишите боту [@userinfobot](https://t.me/userinfobot) — он пришлёт ваш `id`.

### 3. Добавить бота в канал

- Добавьте бота **администратором** вашего канала (права: читать сообщения).

### 4. Загрузить код на GitHub

```bash
git clone https://github.com/ВАШ_ЛОГИН/giveaway-bot
# или создайте новый репозиторий и загрузите файлы
```

Структура репозитория:
```
giveaway-bot/
├── bot.py
├── requirements.txt
└── .github/
    └── workflows/
        └── bot.yml
```

### 5. Добавить секреты в GitHub

Перейдите: **Settings → Secrets and variables → Actions → New repository secret**

| Название секрета | Значение | Пример |
|---|---|---|
| `BOT_TOKEN` | Токен от BotFather | `123456:ABC-DEF...` |
| `CHANNEL_ID` | Username или ID канала | `@mychannel` или `-1001234567890` |
| `ADMIN_ID` | Ваш Telegram ID (число) | `987654321` |
| `GIVEAWAY_TITLE` | Название розыгрыша | `Розыгрыш iPhone 15!` |
| `PRIZE_TEXT` | Описание приза | `iPhone 15 Pro 256GB` |

### 6. Запустить бота

**Actions → Run Giveaway Bot → Run workflow**

Бот запустится и будет работать ~5 часов, затем автоматически перезапустится через cron.

---

## 📋 Команды

### Для всех:
| Команда | Описание |
|---|---|
| `/start` | Главное меню розыгрыша |

### Только для администратора:
| Команда | Описание |
|---|---|
| `/admin` | Панель администратора |
| `/announce` | Отправить объявление победителя в канал |
| `/reset` | Сбросить розыгрыш (удалить всех участников) |

---

## 🛠 Панель администратора

В `/admin` доступны:
- 👥 **Список участников** — постраничный список всех записавшихся
- 🎲 **Выбрать победителя** — случайно или вручную из списка
- 🔒 **Завершить/открыть** розыгрыш

---

## 💾 Хранение данных

Участники сохраняются в `participants.json`.  
Файл передаётся между запусками через **GitHub Actions Artifacts** (хранится 90 дней).

---

## ⚠️ Важно

- GitHub Actions **бесплатен** для публичных репозиториев (2000 мин/мес для приватных).
- Бот перезапускается каждые 5 часов — это нормально, участники не теряются.
- Не публикуйте `BOT_TOKEN` в коде — только через Secrets!
